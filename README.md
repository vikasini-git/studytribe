# StudyTribe – Study Group Finder

## Overview

StudyTribe is a web application that helps students discover, create, and join study groups based on their interests, subjects, and academic goals. 
The platform enables students to collaborate, share resources, and build productive learning communities.

## Features

* User Registration and Login
* Secure Authentication
* Student Dashboard
* Create Study Groups
* Join Existing Study Groups
* View Group Information
* User Profile Management
* Database Integration
* Responsive User Interface

## Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQLite
